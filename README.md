# gerador-senhas-python
um gerador de senhas de 6 a 18 caracteres em python, código limpo e simplificado
Projeto simples para geração de senhas seguras, com validação de entrada e uso de geração criptograficamente segura.

## Funcionalidades
- Geração de senhas aleatórias
- Validação de entrada do usuário
- Uso de biblioteca segura (`secrets`)
- Personalização de tamanho da senha
